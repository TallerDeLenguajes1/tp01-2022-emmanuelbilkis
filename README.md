# Git Ignore 
El git ignore sirve para ignorar los archivos y/o carpetas que no se desea subir a la nube pero que por alguna razon deben estar (o se los desea tener) de forma local en el repositorio. Por ejemplo cuando hay algun archivo con usuarios y contrase√±a q no se desea compartir, tambien cuando hay archivos de configuracion y son innecesarios subirlos.
## Reglas
-Se crea un archivo _.gitignore.txt_ y en este se incluyen todos los archivos y/o carpetas q no se desean tener en cuenta para subir al repositorio remoto.
-Luego se hace un **git add** gitingone.txt 
-Finalmente **git commit** gitignore.txt
